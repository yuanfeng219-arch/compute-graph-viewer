说明

上方是openPangu-2.0-Flash 模型几何投影：轴测视图拖拽旋转、按住 Ctrl/Command 拖拽平移；正视和侧视锁定旋转，直接拖拽即可平移；三种视图均可直接滚轮缩放。同一模型投影从上到下显示 Input / Parallel Embedding、当前 Layer 平面、Final RMSNorm / LM Head / Logits 与 MTP L46–L48 ×3 尾支。模型深度包含 Dense L0–L1 ×2 与 MoE L2–L45 ×44，在侧视中以独立层段展开，并与 PP、mHC state 和通信指标联动。下方 32 卡按 1F1B 流水播放，竖线连到当前算子的卡。

代表配置 DP2 × PP4 × TP2 × EP2 = 32 ranks；这是基于 openpangu_2_0_flash_modelviz.html 的训练通信 demo，保留 TP/EP/PP 通信映射并把 routed experts 改为 256。

通信域：TP/SP AG/RS 固定 DP/PP/EP 坐标，只改变 TP rank；EP dispatch_combine 固定 DP/PP/TP 坐标，只改变 EP rank；PP send/recv 连接相邻 stage 的对应 DP/TP/EP rank；DP gradient sync 固定 PP/TP/EP 坐标，只改变 DP replica。Shared Expert 与 routed + shared add 均为本地计算，不属于 EP collective。

证据口径：本页逐 Layer 数值均为 simulated 演示数据，不冒充 profiler 实测。侧视图四条蓝线表示同一 Layer-boundary mHC state 的 4 支 stream；每个 Xl hover 会列出 residual / activation / dHidden 与完整 D/PP/TP/EP/rank、local/logical shape。三条数值轨道的浅色阴影表示 rank group p10–p90。通信轨道只画 Exposed Comm，同一 PP stage 内显示 —；Active、Wait、Exposed 在 tooltip 与泳道中分开。

语义依据：Megatron 0.16 并行指南、Megatron MoE 指南、Megatron P2P API。
